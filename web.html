Colorful HTML Xylophone with Web Audio

Click or press keys (1–8) to play notes. Visual feedback shows struck keys. This uses the Web Audio API to synthesize tones with frequency-based envelopes.

How it works

  • Sound: Each key plays a sine-like tone with a short envelope to resemble a xylophone note.
  • Feedback: The pressed key gains an active visual state for immediate feedback.
  • Interactivity: You can also use the number keys 1–8 to trigger notes.
